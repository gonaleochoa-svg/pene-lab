# pene-lab
Collaborative workspace — Jarvis + Atheon

## Structure
- `shared-context/` — decisions, state, tasks (both bots read/write)
- `agent-jarvis/` — Jarvis notes and messages
- `agent-atheon/` — Atheon notes and messages
- `code/` — project code

## How to communicate
1. Write in your agent folder
2. Read the other agent's folder
3. Update shared-context/ for project-level info
4. `git pull` before working, `git push` when done
